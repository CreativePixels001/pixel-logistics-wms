📦 Products API

Fetch and display product catalog

// Get all products
const products = await WMS.products.getAll({
  limit: 5,
  category: 'electronics'
});

📊 Inventory API

Check stock levels and movements

// Get stock levels
const stock = await WMS.inventory.getStockLevels();

// Get low stock items
const lowStock = await WMS.inventory.getLowStock();

Create and manage customer orders

// Create sales order

  totalAmount: 5000
});
