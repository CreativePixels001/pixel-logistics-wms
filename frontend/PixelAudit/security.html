Security at PixelAudit

Your data security is our top priority

Enterprise-Grade Security

PixelAudit implements industry-standard security practices to protect your audit data, client information, and sensitive photos. We take security seriously, even at our low price point.

Security Measures

Data Encryption

All data is encrypted in transit using HTTPS/TLS 1.3 and at rest using AES-256 encryption. Your audit data, photos, and client information are always encrypted.

Secure Cloud Storage

Photos are stored on AWS S3 with strict access controls and automatic backups. Database is hosted on MongoDB Atlas with 24/7 monitoring and redundancy.

OAuth 2.0 Authentication

We use Google OAuth for secure authentication. We never store your Google password. JWT tokens are used for session management with automatic expiration.

Access Controls

Role-based access ensures field auditors can only access assigned audits via unique links. Admin users have full control. No one can see data from other accounts.

24/7 Monitoring

Our infrastructure is monitored 24/7 for suspicious activity. Automated alerts notify us of any potential security issues within seconds.

Regular Backups

Automatic daily backups with 30-day retention. In case of data loss, we can restore from any backup point within the last month.

Compliance & Certifications

We are working towards the following certifications:

  • GDPR Compliance: European data protection standards
  • ISO 27001: Information security management (planned for 2026)
  • SOC 2 Type II: Security controls audit (planned for 2026)

Responsible Disclosure

If you discover a security vulnerability, please report it to security@pixelaudit.com. We take all reports seriously and will respond within 24 hours. We do not currently offer a bug bounty program but appreciate responsible disclosure.
