Planned Endpoints

GET /api/v1/audits

List all audits with filtering and pagination

POST /api/v1/audits

Create and assign a new audit

GET /api/v1/audits/:id

Get audit details including all responses

GET /api/v1/audits/:id/pdf

Download PDF report for completed audit

POST /api/v1/clients

Create a new client

GET /api/v1/templates

List all available audit templates

Authentication

All API requests require authentication via Bearer token: 

Authorization: Bearer your_api_key_here

Rate Limits

  • Free Tier: Not available
  • ₹1 Tier: Not available
  • ₹100 Tier: Not available
  • ₹1,000 Enterprise: 1,000 requests/hour
