Getting Started

Welcome to PixelNotes! This guide will help you get up and running in minutes.

1. Sign Up or Try Demo

  1. Visit PixelNotes

    Go to login page to access PixelNotes

  2. Choose Your Method

    Click "Sign in with Google" for full features, or "Try Demo Mode" to test without login

  3. Complete Setup

    If using Google, authorize access. You'll be redirected to your dashboard automatically

  4. Start Creating

    Click "New Note" to create your first note

💡 Tip: Demo Mode gives you full access to all features without requiring sign-up. Your notes are saved locally in your browser. Sign in with Google to sync across devices.

Creating & Managing Notes

Creating a New Note

There are multiple ways to create notes:

  • Dashboard: Click the "New Note" button
  • Editor: Click "+" icon in the notes list
  • Keyboard: Press Ctrl+N (Windows) or Cmd+N (Mac)

Note Structure

Each note in PixelNotes has three main components:

  • Title: Give your note a descriptive name
  • Category/Tags: Organize with categories and tags
  • Content: The main note body with rich text support

Rich Text Formatting

PixelNotes supports various formatting options:

  • Bold, italic, underline text
  • Bulleted and numbered lists
  • Headers and subheaders
  • Code blocks for technical content
  • Links and references
📝 Note: All changes are auto-saved every few seconds. You'll see a "Saved" indicator when changes are synced.

Multi-Language Voice Input

Starting Voice Input

  1. Click Microphone Icon

    Located in the editor toolbar

  2. Select Language

    Choose from 10+ supported languages including English, Hindi, Marathi, Tamil, Telugu, etc.

  3. Grant Permission

    Allow browser to access your microphone (one-time permission)

  4. Start Speaking

    Speak naturally. Your words will appear in real-time

Voice Input Best Practices

  • Clear Environment: Use in a quiet space for best accuracy
  • Natural Speech: Speak at your normal pace, no need to slow down
  • Punctuation: Say "comma", "period", "question mark" for punctuation
  • New Line: Say "new line" or "new paragraph"
  • Code-Switching: Mix languages naturally - the system adapts
💡 Pro Tip: For meeting notes, start voice input at the beginning of the meeting and let it run. PI will automatically add timestamps and structure the content.

PI (Pixel Intelligence) Assistant

What is PI?

PI is your intelligent note-taking companion that learns from your writing, suggests improvements, and helps you extract more value from your notes.

Key PI Features

1. Real-Time Suggestions

As you type, PI analyzes your content and provides context-aware suggestions:

  • Related notes and topics
  • Potential action items
  • Relevant categories and tags
  • Writing improvements

2. Auto-Summarization

Generate summaries of long notes instantly:

  1. Click the "Summarize" button in PI panel
  2. PI analyzes your note content
  3. Receive a concise summary with key points
  4. Copy or insert summary into note

3. Action Item Extraction

PI automatically identifies tasks and deadlines in your notes:

  • Detects action verbs (do, complete, send, call, etc.)
  • Identifies due dates and time references
  • Creates a todo list from meeting notes
  • Highlights important decisions

4. Smart Tags

PI suggests relevant tags based on note content, making organization effortless.

Example: If you write "Call John about Q4 sales report by Friday"

PI suggests:
- Action: Call John
- Topic: Sales report
- Deadline: This Friday
- Tags: Sales, Q4, Follow-up

Smart Sections

Understanding Sections

Smart Sections automatically organize your notes chronologically, making long-running projects and journals easy to navigate.

How Sections Work

  • Auto-Creation: Sections are created automatically based on date (weekly or monthly)
  • Collapsible: Click section headers to expand/collapse content
  • Visual Timeline: See your note history at a glance
  • Easy Navigation: Jump to any time period quickly

Using Sections Effectively

  1. Enable Sections

    Toggle "Use Sections" in note settings

  2. Choose Period

    Select Weekly or Monthly sections based on your needs

  3. Add Content

    New entries automatically appear in the current period's section

  4. Navigate History

    Collapse old sections to focus on current work while preserving history

📌 Use Cases: Sections are perfect for: Daily journals, Project logs, Meeting series, Customer interaction history, Development notes, Personal growth tracking

Search & Find

Basic Search

Find notes quickly using the search bar:

  • Type any keyword or phrase
  • Results appear instantly as you type
  • Matches in title, content, and tags are highlighted

Advanced Search

Use filters to narrow down results:

  • Date Range: Find notes from specific time periods
  • Categories: Search within specific categories
  • Tags: Filter by single or multiple tags
  • Sort Options: By date, relevance, or title

Search Tips

Search operators:
- Use quotes for exact phrases: "project meeting"
- Use AND to require multiple terms: sales AND Q4
- Use OR for alternatives: report OR summary
- Use - to exclude: meeting -cancelled
🔍 Pro Tip: PI's semantic search understands context. Searching "customer complaints" will also find notes about "client issues" or "user feedback".

Team Collaboration

Sharing Notes

  1. Open Note

    Navigate to the note you want to share

  2. Click Share

    Click the share icon in the toolbar

  3. Add Team Members

    Enter email addresses of collaborators

  4. Set Permissions

    Choose View Only or Edit access

Team Features

  • Shared Templates: Create note templates for consistent team documentation
  • Comments: Add comments to specific sections (Enterprise plan)
  • Version History: See who made what changes and when
  • Team Dashboard: Overview of shared notes and team activity
⚠️ Important: Team collaboration features require a Team or Enterprise plan. Demo Mode does not support sharing.

Keyboard Shortcuts

Global Shortcuts

  • Ctrl+N / Cmd+N - New Note
  • Ctrl+S / Cmd+S - Save (manual save)
  • Ctrl+F / Cmd+F - Search
  • Ctrl+K / Cmd+K - Quick Commands

Editor Shortcuts

  • Ctrl+I / Cmd+I - Italic
  • Ctrl+U / Cmd+U - Underline
  • Ctrl+L / Cmd+L - Insert Link

Navigation Shortcuts

  • Ctrl+← / Cmd+← - Previous Note
  • Ctrl+→ / Cmd+→ - Next Note

  • Ctrl+↓ / Cmd+↓ - Scroll to Bottom

Export & Backup

Exporting Notes

Export individual notes or entire collections:

  • Markdown: Plain text format with formatting preserved
  • PDF: Professional document format
  • HTML: Web-ready format with full styling
  • JSON: Raw data for custom processing

Backup Options

  1. Automatic Backup

    Enterprise plans include automatic daily backups to Google Drive

  2. Manual Export

    Go to Settings → Export → Select format and notes

  3. Download Backup

    Complete archive downloads as .zip file

🔄 Backup Strategy: We recommend exporting important notes weekly and storing backups in multiple locations (local drive, cloud storage, external drive).

Troubleshooting

Common Issues

Voice Input Not Working

  • Check microphone permissions in browser settings
  • Ensure HTTPS connection (required for microphone access)
  • Try a different browser (Chrome recommended)
  • Check if another app is using the microphone

Notes Not Syncing

  • Check internet connection
  • Refresh the page (Ctrl+R / Cmd+R)
  • Clear browser cache and cookies
  • Sign out and sign back in

PI Not Responding

  • Ensure note has enough content (minimum 50 words)
  • Wait a few seconds for AI processing
  • Toggle PI panel off and on
  • Check if ad-blocker is interfering

Search Not Finding Notes

  • Check spelling of search terms
  • Try different keywords or synonyms
  • Remove filters to broaden search
  • Wait for search index to update (may take a minute)
